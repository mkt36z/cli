# Marketing OKR Framework

## Purpose
A structured framework for setting, tracking, and reviewing marketing Objectives and Key Results. Designed to align marketing activities with business outcomes and ensure every team member knows what matters most.

---

## How to Write Good Marketing OKRs

### The Rules

1. **3-5 Objectives per quarter** — fewer is better
2. **2-4 Key Results per Objective** — measurable, time-bound, ambitious
3. **Objectives are qualitative** — they describe the desired state ("Become the go-to resource for...")
4. **Key Results are quantitative** — they prove you achieved the objective (numbers, percentages, yes/no)
5. **70% achievement = healthy ambition** — if you hit 100% every quarter, your OKRs are too easy
6. **OKRs cascade** — company → marketing → individual

### Objective Formula
```
[Action verb] + [what] + [why it matters]
Example: "Establish Marketing36z as the default marketing OS for solo founders"
```

### Key Result Formula
```
[Verb] [metric] from [X] to [Y] by [date]
Example: "Increase organic signups from 200/mo to 500/mo by end of Q2"
```

## OKR Review Cadence

### Weekly Check-In (10 min)

| Question | Answer |
|----------|--------|
| **KRs on track:** | List KRs in green |
| **KRs at risk:** | List KRs in yellow/red |
| **Blockers:** | What's preventing progress? |
| **This week's focus:** | Top 1-2 actions to move KRs |

### Monthly Review (30 min)

| Question | Answer |
|----------|--------|
| **Progress update for each KR:** | Score 0-1.0 |
| **What's working:** | Double down |
| **What's not working:** | Adjust or kill |
| **Resource reallocation needed?** | Yes/No + details |
| **New information that changes strategy:** | |

### Quarterly Retrospective (60 min)

| Question | Answer |
|----------|--------|
| **Final score for each KR:** | 0-1.0 |
| **Overall objective score:** | Average of KRs |
| **What did we learn?** | |
| **What will we do differently next quarter?** | |
| **OKRs to carry forward:** | |
| **OKRs to retire/replace:** | |

---

## Scoring Guide

| Score | Meaning | Action |
|:-----:|---------|--------|
| 0.0 - 0.3 | Failed to make meaningful progress | Investigate why — wrong goal, wrong approach, or under-resourced? |
| 0.4 - 0.6 | Made progress but fell short | Review what worked, adjust for next quarter |
| 0.7 - 0.8 | Strong performance, healthy stretch | Sweet spot — this is where you want most OKRs to land |
| 0.9 - 1.0 | Crushed it | Celebrate, then ask: were OKRs ambitious enough? |

## Common OKR Mistakes

| Mistake | Fix |
|---------|-----|
| Key Results are tasks, not outcomes | "Launch email campaign" → "Generate 200 leads from email campaign" |
| Too many OKRs (>5 objectives) | Ruthlessly prioritize — if everything is a priority, nothing is |
| No baseline measurement | You can't track improvement without a starting point |
| OKRs set in isolation | Align with company OKRs and adjacent teams (sales, product) |
| Never reviewing mid-quarter | Weekly check-ins prevent quarter-end surprises |
| Tying OKRs to compensation | OKRs should encourage ambition, not sandbagging |
