# Workflow: Client Onboarding (Agency)

## Purpose
A structured workflow for agencies to onboard new clients efficiently, extract the strategic inputs needed for Marketing36z OS to produce great work, and set expectations that prevent scope creep and churn.

## When to Use This Workflow
- New agency client signed
- Existing client starting a new engagement
- Re-onboarding a returning client
- Orchestrator routes here for agency "new client" requests

---

## Phase 1: Pre-Onboarding Preparation (Before Kickoff)

### Step 1.1: Internal Setup

| Task | Owner | Deadline | Status |
|------|-------|:--------:|:------:|
| Create client folder (shared drive + project management) | Account Manager | Day -3 | |
| Assign team members and roles | Account Director | Day -3 | |
| Review signed SOW — scope, deliverables, timeline | Account Manager | Day -2 | |
| Send welcome email with kickoff agenda | Account Manager | Day -2 | |
| Prepare onboarding questionnaire | Strategist | Day -2 | |
| Set up client communication channel (Slack, Teams) | Account Manager | Day -1 | |
| Configure Marketing36z OS with client brand voice | Strategist | Day -1 | |

### Step 1.2: The Client Welcome Email

```
Subject: Welcome to [Agency] — Here's What Happens Next

Hi [Client Name],

Welcome aboard! We're thrilled to work with you.

Here's what to expect:

1. **Onboarding Questionnaire** (attached) — Please complete before our kickoff
   call. This helps us hit the ground running.

2. **Kickoff Meeting** — Scheduled for [date/time]. We'll cover:
   - Your goals and success metrics
   - Brand guidelines and voice
   - Communication cadence and approval process
   - 90-day plan overview

3. **Access Needed** — Please share:
   - Analytics access (GA, social, ads)
   - CMS / website access
   - Existing brand assets (logos, fonts, guidelines)
   - Past campaign data / reports

Looking forward to our kickoff!
```

## Phase 2: Discovery and Strategic Foundation (Week 1)

### Step 2.1: Client Onboarding Questionnaire

**Section A: Business Foundation**

| Question | Client Response |
|----------|----------------|
| What is your core product/service? | |
| Who is your ideal customer? (Be specific: role, company size, industry) | |
| What problem do you solve for them? | |
| What's your current revenue/ARR? | |
| What's your revenue target for this engagement? | |
| Who are your top 3 competitors? | |
| Why do customers choose you over alternatives? | |
| What's your pricing model and price points? | |

**Section B: Marketing Current State**

| Question | Client Response |
|----------|----------------|
| What marketing are you currently doing? | |

| What tools/platforms do you use? | |
| Do you have a content library? Where? | |
| What does your sales process look like? | |

|----------|----------------|

| What words/phrases should we NEVER use? | |

| Who needs to approve content before publishing? | |
| What's the typical approval turnaround time? | |

**Section D: Goals and Success Metrics**

| Question | Client Response |
|----------|----------------|
| What does success look like in 90 days? | |
| What does success look like in 6 months? | |
| What are the 3 most important KPIs? | |
| What's the one metric that matters most right now? | |

| Time | Topic | Led By |

### Step 2.3: Post-Kickoff: Marketing36z OS Configuration

| Configuration | Input Source | System Impact |
|--------------|-------------|-------------|
| Brand voice adjectives | Questionnaire Section C | Content Engine voice matching |

| Competitor list | Questionnaire Section A | Competitive analysis inputs |

| Channel priorities | Current state analysis | Channel Operator focus areas |

## Phase 4: Operating Cadence (Ongoing)

### Communication Framework

| Touchpoint | Frequency | Format | Agenda |
|-----------|:---------:|--------|--------|
| **Status update** | Weekly | Email / async | Deliverables completed, upcoming, blockers |
| **Check-in call** | Bi-weekly | 30 min video | Review metrics, align on priorities |
| **Monthly report** | Monthly | Slide deck | Full performance review, recommendations |
| **QBR** | Quarterly | 60 min video | Strategic review, goal adjustment, renewal |

### The Approval Process

| Content Type | Approval Required? | Turnaround SLA | Escalation |
|-------------|:------------------:|:--------------:|-----------|
| Social posts | Client review | 48 hours | Auto-publish if no response in 72h |
| Blog posts | Client approval | 72 hours | Escalate to Account Director |
| Email campaigns | Client approval | 72 hours | Hold until approved |
| Paid ad creative | Client approval | 48 hours | Delays reduce budget efficiency |
| Strategy documents | Client approval | 5 business days | Schedule review meeting |

## Phase 5: Onboarding Success Metrics

### The 30-60-90 Onboarding Scorecard

| Milestone | Day 30 | Day 60 | Day 90 |
|-----------|--------|--------|--------|
| **Strategic foundation** | Audit complete, 90-day plan approved | — | — |
| **Content engine** | Calendar built, first content published | Content cadence established | Content pipeline self-sustaining |
| **Campaign launch** | Quick win campaign live | Primary campaign optimized | Full campaign suite running |
| **Reporting** | Baseline metrics documented | First monthly report delivered | QBR completed |
| **Relationship** | Communication cadence established | Client satisfaction survey (target: 8+/10) | Renewal/expansion discussion |

### Client Health Score

| Dimension | Score (1-5) | Weight | Weighted Score |
|-----------|:-----------:|:------:|:--------------:|
| Engagement (responsiveness, attendance) | | 25% | |
| Satisfaction (feedback, NPS) | | 30% | |
| Results (KPI performance vs. targets) | | 30% | |
| Growth potential (expansion, referral) | | 15% | |
| **Total** | | | **/5** |

**Score < 3.0:** At-risk. Immediate account review.
**Score 3.0-4.0:** Healthy but needs attention.
**Score 4.0+:** Strong. Explore expansion and referral.

## Integration with Marketing36z OS

| System Component | How This Workflow Connects |
|-----------------|---------------------------|
| **Orchestrator** | Routes new client requests through onboarding before any execution |
| **Strategy Planner** | Receives ICP, competitor, and positioning inputs from onboarding |
| **Content Engine** | Configured with brand voice from onboarding questionnaire |
| **Quality Guardian** | Calibrated with client-specific quality criteria |
| **Multi-Brand Management** | Each client creates a new brand profile in policies.yaml |
| **QBR Framework** | QBR template used at 90-day mark and quarterly thereafter |
