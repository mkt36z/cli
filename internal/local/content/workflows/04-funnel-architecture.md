# Workflow: Funnel Architecture

## Purpose
Design the complete customer journey architecture — from first touch to lifetime customer — using proven funnel and value ladder frameworks. This workflow ensures every campaign exists within a coherent system that maximizes customer lifetime value, not just first-transaction revenue.

## When to Use This Workflow
- Founder is designing their go-to-market for the first time
- Revenue exists but there's no systematic customer journey
- Average order value is too low (no upsell/cross-sell architecture)
- Customer churn is high (no retention/ascension path)
- Strategy Planner recommends funnel architecture before campaign execution

---

## Phase 1: Value Ladder Design (Brunson Framework)

### The Concept
A value ladder is a series of offers at ascending price points, each delivering more value. Customers ascend naturally when each level delivers a result that creates desire for the next level.

### The Standard Value Ladder Tiers

```
                    ┌─────────────┐
                    │  Tier 5:    │  $10K-100K+
                    │  High-Touch │  (Done-For-You / Partnership)
                    │  Premium    │
                  ┌─┴─────────────┴─┐
                  │  Tier 4:        │  $1K-10K
                  │  Core Offer     │  (Done-With-You / Coaching)
                  │  (Primary Rev)  │
                ┌─┴─────────────────┴─┐
                │  Tier 3:            │  $100-1K
                │  Entry Product      │  (Course / Template / Tool)
                │  (Self-Liquidating) │
              ┌─┴─────────────────────┴─┐
              │  Tier 2:                │  $1-100
              │  Tripwire / Low-Ticket  │  (Mini-course / Trial / Template)
              │  (Buyer Conversion)     │
            ┌─┴─────────────────────────┴─┐
            │  Tier 1:                    │  FREE
            │  Lead Magnet / Free Value   │  (Guide / Tool / Assessment / Trial)
            │  (List Building)            │
            └─────────────────────────────┘
```

### For Each Tier, Define:

| Element | Description |
|---------|------------|
| **Name** | Outcome-oriented name (see Offer Design Lab naming formula) |
| **Price** | Specific price point |
| **Dream Outcome** | What transformation does this tier deliver? |
| **Delivery** | How is it delivered? (Delivery Cube from Offer Design Lab) |
| **Ascension Trigger** | What result or realization makes them want the NEXT tier? |
| **Time to Value** | How quickly do they see results at this tier? |
| **Margin** | What's the profit margin? |
| **Purpose** | Lead gen / buyer conversion / core revenue / profit maximizer / LTV |

### Value Ladder Rules
1. Each tier must deliver a COMPLETE result (not an incomplete one that forces an upgrade)
2. Each tier naturally creates desire for the next (the result at this level reveals the next problem)
3. Price jumps between tiers should be 3-10x (not 100x — that gap is too wide)
4. The lowest tier should be accessible to anyone in your ICP
5. The highest tier should be aspirational but achievable for your best customers

## Phase 2: Funnel Types & Selection

### Funnel Type 1: Lead Magnet Funnel (List Building)
**Purpose:** Convert strangers into leads

```
Traffic → Landing Page → Lead Magnet Delivery → Nurture Sequence → Core Offer
```

**When to use:** You need more leads. Always-on, foundational funnel.

**Key elements:**
- Lead magnet solves ONE specific problem (from Phase 1 problem list)
- Landing page optimized for single conversion
- Instant delivery (email or in-app)
- Follow-up: Soap Opera Sequence (see Email Sequences below)

### Funnel Type 2: Self-Liquidating Offer (SLO) Funnel
**Purpose:** Acquire customers at breakeven, monetize on the backend

```
Ad → Landing Page → Tripwire ($7-47) → Order Bump (+$27-97) → Upsell 1 → Upsell 2 → Thank You
```

**When to use:** You have a proven offer and want to scale with paid ads.

**Key elements:**
- Tripwire price covers ad cost (self-liquidating)
- Order bump at checkout (30-40% take rate typical)
- One-click upsell immediately after purchase (10-15% take rate)
- Second upsell or downsell (5-10% take rate)
- Goal: $0 cost per customer acquisition (ads paid for by front-end revenue)

### Funnel Type 3: Webinar Funnel
**Purpose:** Sell mid-to-high-ticket offers ($500-$5,000)

```
Registration Page → Confirmation → Webinar → Application/Order Page → Follow-up Sequence
```

**When to use:** Offer requires education before purchase. Complex products/services.

**Key elements:**
- Registration page with strong hook + 3 curiosity bullets
- Webinar structure: Hook (10min) → Story (10min) → Content/Teaching (30min) → Transition → Offer (20min) → Close (10min)
- "Epiphany Bridge" storytelling (see Content Engine)
- Stack slide for value presentation
- 3-day follow-up sequence post-webinar (urgency-based)

### Funnel Type 4: Application Funnel
**Purpose:** Sell high-ticket offers ($5,000+)

```
Case Study/VSL → Application Page → Call Booking → Sales Call → Onboarding
```

**When to use:** Premium offers that require qualification.

**Key elements:**
- VSL or case study page to pre-sell
- Application form to qualify and create commitment
- Call booking with confirmation sequence
- Sales call script (consultative, not pushy)
- Onboarding sequence post-purchase

### Funnel Type 5: Challenge Funnel
**Purpose:** Build community, prove value through action, sell at the end

```
Registration → 5-7 Day Challenge → Daily Emails + Community → Graduation Offer
```

**When to use:** Audience needs to experience a quick win before committing.

**Key elements:**
- 5-7 days of daily action steps with small wins
- Community (Facebook group, Slack, Discord) for accountability
- Daily live calls or videos
- Day 5-7: Present core offer as the "next step" after the challenge

### Funnel Selection Matrix

| Founder Stage | Primary Funnel | Secondary Funnel | Price Range |
|--------------|---------------|-----------------|-------------|
| Pre-revenue, validating | Lead Magnet | Challenge | Free → $47 |
| Early revenue (<$10K/mo) | SLO + Lead Magnet | Webinar | $7 → $997 |
| Growing ($10K-50K/mo) | Webinar + SLO | Application | $47 → $5,000 |
| Scaling ($50K+/mo) | Application + SLO | All running | $47 → $25,000+ |

## Phase 3: Email Sequence Architecture

### Sequence 1: Soap Opera Sequence (Brunson)
**Purpose:** Convert new leads into buyers during the first 5-7 days.
**Timing:** Triggered immediately after opt-in.

| Day | Email | Purpose |
|-----|-------|---------|
| 0 | Delivery + Hook | Deliver lead magnet. Open a story loop about your discovery/transformation |
| 1 | Backstory + Drama | Share the journey. Build empathy. Show "rock bottom" moment |
| 2 | Epiphany | The breakthrough moment. Introduce your unique mechanism |
| 3 | Hidden Benefits | Reveal unexpected benefits beyond the obvious. Build desire |
| 4 | Urgency + CTA | Close the story. Make the offer. Create urgency to act now |
| 5 | Last Chance | Final push. Strongest proof + risk reversal + deadline |

**Key rules:**
- Each email ends with a cliffhanger that compels opening the next
- Written in first person, story format
- Progressive disclosure: each email reveals more, building anticipation
- CTA appears from Day 2 onward, but soft until Day 4

### Sequence 2: Seinfeld Sequence (Brunson)
**Purpose:** Ongoing engagement AFTER the Soap Opera Sequence.
**Timing:** Starts after Day 5-7, runs 2-3x per week indefinitely.

**Format:** Each email is a standalone story or insight that entertains, teaches, and connects back to your offer. Like a Seinfeld episode — it's "about nothing" but always entertaining.

**Content categories to rotate:**
1. Personal story with business lesson
2. Customer success story
3. Common mistake and how to fix it
4. Behind-the-scenes of your business
5. Contrarian industry opinion
6. "What I'd do if I had to start over"
7. Direct promotional email (1 in every 4-5)

### Sequence 3: Launch Sequence (Pre-Launch → Launch → Close)

| Phase | Duration | Focus |
|-------|----------|-------|
| Pre-launch (Seed) | 7-14 days | Tease the offer. Build anticipation. Gather feedback. |
| Pre-launch (Build) | 3-7 days | Share the story behind the offer. Social proof. Waitlist. |
| Launch (Open) | Day 1 | Announce the offer. Full value stack. Guarantee. Early-bird bonus. |
| Launch (Sustain) | Days 2-5 | Case studies, FAQ, objection handling, bonus reminders |
| Launch (Close) | Days 6-7 | Urgency. Final testimonials. Deadline. Cart close emails (3 on last day) |

### Sequence 4: Re-Engagement Sequence
**Trigger:** 30 days of no email opens or clicks.

| Email | Strategy |
|-------|----------|
| 1 | Subject: "Should I stop emailing you?" — Direct, honest, re-permission |
| 2 | Subject: "I noticed you've been quiet" — Offer a different content format |
| 3 | Subject: "Last email from me (unless)" — Final chance, compelling reason to stay |
| | If no engagement → remove from active list (improves deliverability) |

## Phase 4: Customer Ascension Architecture (Abraham × Hormozi)

### Jay Abraham's 3 Growth Levers Applied

| Lever | Strategy | Funnel Element |
|-------|----------|----------------|
| **More customers** | Lead magnets, content, ads, referrals | Front of Value Ladder |
| **Higher transaction value** | Upsells, order bumps, premium tiers | Middle of Value Ladder |
| **More frequent transactions** | Subscriptions, repurchases, renewal offers | Continuity + Retention |

### Ascension Triggers
Define what event moves a customer from one tier to the next:

| Transition | Trigger | Automated Action |
|-----------|---------|-----------------|
| Free → Tripwire | Consumed lead magnet + visited pricing 2x | Send tripwire offer email |
| Tripwire → Core | Completed tripwire + opened 5+ emails | Send webinar invite or sales page |
| Core → Premium | Used product 30+ days + achieved first result | Send case study + upgrade offer |
| Premium → High-touch | Renewed 2x + referred 1+ customer | Personal outreach with VIP offer |

### Retention Loop Design

```
Onboarding (Days 1-7)
      │
      ▼
Quick Win Achieved (Day 3-14)
      │
      ▼
Habit Formation (Days 14-30)
      │
      ▼
Milestone Celebration (Day 30, 60, 90)
      │
      ▼
Community Integration (Ongoing)
      │
      ▼
Referral Prompt (After first big result)
      │
      ▼
Ascension Offer (After 2+ milestones)
```

## Phase 5: Traffic Architecture (Dream 100 Strategy)

### The Dream 100 Concept (Brunson)
Identify the 100 people, platforms, and places where your dream customers already congregate.

**Categories:**
1. **People:** Influencers, thought leaders, podcast hosts your ICP follows
2. **Platforms:** Websites, communities, forums, groups where your ICP hangs out
3. **Publications:** Newsletters, blogs, media outlets your ICP reads

### Earning vs. Buying Your Way In

| Strategy | Earned (Free) | Bought (Paid) |
|----------|--------------|---------------|
| People | Guest on their podcast, write for their blog, co-create content | Sponsor their content, pay for promotion, influencer deal |
| Platforms | Create native content, become active community member, SEO | Run ads on the platform, sponsor events, paid placement |
| Publications | Pitch guest articles, provide expert quotes, write case studies | Sponsored content, newsletter ads, display advertising |

**Process:**
1. List 100 targets across all three categories
2. Follow/subscribe to all of them
3. Engage authentically for 30 days (comments, shares, genuine value)
4. Make your first "ask" after building relationship (not before)
5. Provide asymmetric value — give them something more valuable than what you're asking for

6. Dream 100 target list framework
7. Traffic allocation plan (earned vs. bought, by channel)
8. Revenue model projection (front-end, back-end, LTV)
