## The Core Principle

> A brand is not a logo. A brand is what people say about you when you're not in the room.

Your brand foundation defines WHO you are, WHY you matter, and HOW you show up — consistently, across every touchpoint. Without this foundation, every piece of marketing is a one-off that builds nothing.

## Part 2: Brand Voice

### The Voice Framework

Your brand voice is defined by 3-5 adjectives + what each means and doesn't mean.

| Voice Adjective | What It Means | What It Doesn't Mean | Example |
|----------------|--------------|---------------------|---------|
| e.g., **Direct** | We get to the point. Short sentences. No fluff. | We're not rude or dismissive. | "Here's the framework. 3 steps." NOT "We'd like to share some thoughts..." |
| e.g., **Confident** | We speak with authority because we've done the work. | We're not arrogant or condescending. | "This works. Here's proof." NOT "We believe we might be able to help..." |
| | | | |
| | | | |
| | | | |

### Voice Consistency Checklist

Test any content against these questions:
- [ ] Does it sound like the same person/brand as our last 5 pieces?
- [ ] Would our ICP feel this was written FOR them?
- [ ] Does it match our personality spectrum positions?
- [ ] Could a competitor say the exact same thing? (If yes → rewrite)
- [ ] Would our "anti-values" be horrified? (If no → not distinctive enough)

### The Do / Don't Voice Guide

| We DO | We DON'T |
|-------|----------|
| Use contractions (we're, you'll, it's) | Sound like a legal document |
| Address the reader as "you" | Say "one might consider" |
| Use short paragraphs (1-3 sentences) | Write walls of text |
| Share opinions and take positions | Hedge with "maybe" and "perhaps" |
| Use concrete examples and numbers | Make vague, generic statements |
| Acknowledge complexity honestly | Oversimplify or overpromise |

## Part 3: Brand Messaging

### The Messaging Hierarchy

```
┌────────────────────────────────────────────┐
│  BRAND PROMISE (One sentence)               │
│  The single most important thing we deliver │
├────────────────────────────────────────────┤
│  POSITIONING STATEMENT                      │
│  What makes us different and why it matters │
├────────────────────────────────────────────┤
│  MESSAGING PILLARS (Max 3)                  │
│  The proof categories that support our      │
│  promise                                    │
├────────────────────────────────────────────┤
│  PROOF POINTS                               │
│  Specific evidence under each pillar        │
├────────────────────────────────────────────┤
│  CUSTOMER LANGUAGE BANK                     │
│  Exact words our ICP uses                   │
└────────────────────────────────────────────┘
```

### Brand Promise

```
For [target customer]
who [situation/need],
[Brand] is the [category]
that [key benefit]
because [reason to believe].
```

### Positioning Statement (Dunford-Aligned)

| Element | Your Position |
|---------|--------------|
| **Competitive alternatives:** | What would customers do without you? |
| **Unique attributes:** | What do you have that alternatives don't? |
| **Value:** | What do those attributes mean for the customer? |
| **Target customers:** | Who cares most about that value? |
| **Market category:** | What frame of reference helps them understand you? |

### The Elevator Pitch

**10-Second Version:**
```
"We help [audience] [achieve outcome] without [biggest objection]."
```

**30-Second Version:**
```
"[Audience] struggle with [problem]. Most [current approach] doesn't work
because [root cause]. [Brand] [unique approach] so they can [outcome].
[Social proof or result]."
```

**60-Second Version:**
```
"You know how [audience] [common frustration]?
Most try to solve it by [current approach], but that fails because [reason].
We built [Brand] to [unique solution]. It works by [mechanism].
[Customer name] used it to [specific result] in [timeframe].
If you're a [ICP descriptor] dealing with [problem], [CTA]."
```

## Part 6: Brand Foundation Checklist

Before considering your brand "defined," verify:

- [ ] Brand purpose statement written
- [ ] 3-5 actionable, differentiating values defined
- [ ] Brand personality spectrum positions chosen
- [ ] Brand archetype identified
- [ ] Voice adjectives defined with examples
- [ ] Do/Don't voice guide created
- [ ] Brand promise statement written
- [ ] Positioning statement complete (Dunford-aligned)
- [ ] 3 messaging pillars with proof points
- [ ] Elevator pitches (10s, 30s, 60s) written
- [ ] Visual guidelines documented (colors, fonts, logo)
- [ ] Touchpoint audit completed
- [ ] Customer Language Bank populated
- [ ] All assets accessible in shared location
