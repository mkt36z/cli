## Part 1: Event Types and When to Use Them

| Event Type | Cost | Reach | Lead Quality | Brand Impact | Best For |
|-----------|:----:|:-----:|:---:|:---:|---------|
| **Webinar** | Low | High | Medium | Medium | Lead gen, education, scale |
| **Virtual summit** | Medium | Very High | Medium | High | Brand awareness, thought leadership |
| **Meetup / community event** | Low | Low | High | High | Community building, local presence |
| **Workshop (in-person)** | Medium | Low | Very High | High | Deep engagement, sales acceleration |
| **Conference (own)** | Very High | Medium-High | High | Very High | Category leadership, brand moat |
| **Conference (sponsor)** | High | Medium | Medium | Medium-High | Industry presence, lead capture |
| **Executive dinner / roundtable** | Medium | Very Low | Very High | High | Enterprise ABM, relationship building |
| **Hybrid (in-person + virtual)** | High | High | Medium-High | High | Maximum reach + intimacy |

## Part 2: Event Strategy Framework

### Step 1: Define Event Objectives

| Objective | Primary Metric | Secondary Metric |
|-----------|---------------|-----------------|
| **Lead generation** | Leads captured | Pipeline generated |
| **Brand awareness** | Attendees + social impressions | Share of voice |
| **Customer engagement** | Customer attendance + NPS | Retention impact |
| **Thought leadership** | Speaker invitations, media coverage | Audience growth |
| **Sales acceleration** | Deals influenced | Sales cycle reduction |
| **Community building** | Active community members | Engagement rate |

### Step 2: Event Planning Timeline

| Timeframe | Activity |
|-----------|---------|
| **T-12 weeks** | Define objectives, budget, format, and date |
| **T-10 weeks** | Secure venue/platform, confirm speakers, create event page |
| **T-8 weeks** | Launch registration, begin promotion |
| **T-6 weeks** | Speaker prep, content finalization |
| **T-4 weeks** | Acceleration push: email, social, ads, partnerships |
| **T-2 weeks** | Final logistics, speaker run-through, attendee communications |
| **T-1 week** | Reminder sequence, last-chance registration push |
| **T-1 day** | Final reminders, tech check |
| **Event day** | Execute, engage, capture |
| **T+1 day** | Thank-you emails, recording distribution |
| **T+1 week** | Lead follow-up, content repurposing |
| **T+2 weeks** | Postmortem, ROI analysis |

## Part 3: Conference Sponsorship Playbook

### Evaluating Sponsorship Opportunities

| Criterion | Score (1-5) | Notes |
|-----------|:-----------:|-------|
| **Audience fit** — Is our ICP attending? | | |
| **Speaker opportunity** — Can we get on stage? | | |
| **Booth traffic** — Will we get meaningful conversations? | | |
| **Brand association** — Does this event elevate our brand? | | |
| **Cost vs. alternatives** — Could we get more leads elsewhere? | | |
| **Competitor presence** — Are competitors sponsoring? | | |

**Rule:** Don't sponsor for the logo placement. Sponsor for the conversations and content opportunities.

### Maximizing Conference ROI

**Before the Conference:**

| Action | Owner | Timeline |
|--------|-------|:--------:|
| Set specific goals (meetings booked, leads captured, talks given) | Marketing | T-6 weeks |
| Pre-schedule meetings with high-value attendees | Sales | T-4 weeks |
| Promote your presence on social media | Marketing | T-3 weeks |
| Prepare booth materials, demos, collateral | Marketing | T-2 weeks |
| Brief the booth team on qualifying criteria and talk tracks | Sales + Marketing | T-1 week |

**At the Conference:**

| Action | Purpose |
|--------|---------|
| Staff the booth with senior people (not just junior) | Decision-maker conversations |
| Run a talk or workshop (not a sales pitch) | Thought leadership + leads |
| Host a side event (dinner, happy hour, breakfast) | Intimate relationship building |
| Live-post content (insights, quotes, photos) | Extend reach beyond attendees |
| Scan badges / collect info for EVERY conversation | Lead capture |

**After the Conference:**

| Action | Timeline | Owner |
|--------|:--------:|-------|
| Personalized follow-up emails (not mass blast) | Day +1-2 | Sales |
| Post event recap on LinkedIn/blog | Day +2-3 | Marketing |
| Score and route leads | Day +3-5 | Marketing Ops |
| Send recordings / resources shared at booth | Day +3-5 | Marketing |
| Debrief: What worked, what didn't, ROI | Day +7-14 | Marketing |

### Event Content Strategy

| Segment | Duration | Format | Purpose |
|---------|:--------:|--------|---------|
| Opening keynote | 20-30 min | Presentation | Set the tone, deliver the big idea |
| Panel discussion | 30-45 min | Moderated panel (3-4 speakers) | Multiple perspectives, audience engagement |
| Breakout / workshop | 45-60 min | Interactive, small group | Hands-on learning, deeper engagement |
| Customer spotlight | 15-20 min | Interview or presentation | Social proof, real-world application |
| Networking | 30-60 min | Structured or unstructured | Relationship building |
| Closing | 10-15 min | Summary + CTA | Call to action, next steps |

### Post-Event Metrics

| Metric | Target | Actual |
|--------|:------:|:------:|
| Leads captured | | |
| MQLs generated | | |
| Pipeline influenced ($) | $ | $ |
| Deals closed (within 90 days) | | |
| Revenue attributed ($) | $ | $ |
| Cost per lead | $ | $ |
| Event ROI | x | x |

### Event ROI Calculation

```
Event ROI = (Revenue attributed to event - Total event cost) / Total event cost

Example:
  Total event cost: $15,000
  Revenue attributed (within 90 days): $75,000
  ROI: ($75,000 - $15,000) / $15,000 = 4.0x
```
